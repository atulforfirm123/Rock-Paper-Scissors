The game tracks the score, displaying the number of wins and losses. Additionally, there's a "Rules" button that displays the game rules.

Installation
To play the game locally, clone the repository and open the index.html file in your preferred web browser.

Usage
Upon opening index.html, the game interface will be displayed.

Click on the "Rules" button to view the game rules.

Choose Rock, Paper, or Scissors by clicking on the corresponding button.

The computer will randomly select its move.

The game will display the result (win, loss, or tie) and update the score accordingly.

Click "Play Again" to reset the game and play another round.

Game Rules
Rock beats Scissors.
Scissors beats Paper.
Paper beats Rock.
Scoring
Each win increases the player's score.
Each loss increases the computer's score.
A tie does not affect the score. -->
